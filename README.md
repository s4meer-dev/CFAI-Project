# Algorithm Benchmark & Complexity Visualization System

A web-based tool to visualize and compare the execution time and time complexity of different algorithms.

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Python

## Project Progress
- **Day 1:** Initial setup and project structure.
- **Day 2:** Implemented Bubble Sort and Merge Sort.
- **Day 3:** Implemented Linear and Binary Search with visualization.
- **Day 4:** Expanded algorithm suite (Quick, Selection, Insertion, Jump) and added detailed measurement controls (Iterations, Data States).
- **Day 5:** Added advanced algorithms (Heap Sort, Shell Sort, Counting Sort, Exponential Search), implemented input validation, and refactored backend logic for better performance tracking.
- **Day 6:** Exposed all algorithms in the frontend dropdown — Heap Sort, Shell Sort, Counting Sort, and Exponential Search are now fully selectable and benchmarkable via the UI.

## Supported Algorithms

### Sorting
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort
- Shell Sort
- Counting Sort

### Searching
- Linear Search
- Binary Search
- Jump Search
- Exponential Search
